================================================================================
CRITICAL DESIGN DECISIONS
================================================================================

1. Patient-Level Splitting Only
   - Prevents tile/patch leakage at the source
   - Enforced through all split classes
   - Automatic validation on every split
   - Serialization for traceability

2. Train-Only Preprocessing Fitting
   - PreprocessingContract immutable after fitting
   - Can't refit on new data to prevent leakage
   - Hash tracks contract changes
   - Full versioning support

3. Bootstrap Confidence Intervals
   - Standard 1000 iterations
   - 95% confidence level (2.5-97.5 percentile)
   - Applied to all metrics consistently
   - Coverage validated in tests

4. Unified Experiment Tracking
   - Abstract base class for multiple backends
   - MLflow primary, W&B secondary, DVC for data
   - Consistent API across backends
   - Reproducibility metadata on every run

5. Publication-Quality Visualization
   - 300 DPI baked into defaults
   - Colorblind palette for accessibility
   - Matplotlib + seaborn for consistency
   - Both PNG and PDF for flexibility
