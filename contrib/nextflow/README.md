# Nextflow Workflow (Alternative)

This is an alternative Nextflow DSL2 implementation of the pipeline.
The primary supported orchestrator is **Snakemake** (see `Snakefile` in repo root).

This Nextflow workflow is provided for teams that prefer Nextflow but is not
actively tested in CI. Cloud profiles (AWS Batch, Google Cloud) contain
placeholder values that must be configured for your environment.
